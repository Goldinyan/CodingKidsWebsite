Möchtest du einen Guide hinzufügen oder verbessern?

1. Erstelle eine neue Datei in guides/
2. Schreibe im Markdown-Format
3. Nutze das Format der bestehenden Guides
4. Teste die Anleitung vor dem Merge
5. Mache einen Pull Request!

Format-Template für Guides:
```
# Guide Titel

## Übersicht
[Was wird gelernt]

## Voraussetzungen
[Was sollte der Nutzer können]

## Schritt 1: ...
[Erklärung]

## Schritt 2: ...
[Erklärung]

## Fertig! 🎉
[Herzlichen Glückwunsch]
```
